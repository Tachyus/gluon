Static memberDescription
Create(seedTypes)
Signature: seedTypes:seq<Type> -> JsonSerializer

Constructs a serializer for a given set of seed types. The serializer can the serialize any type in the transitive closure of the depends relation over the seed types. A record type depends on the types of its fields, a union type on the types of its case fields, and so on.
